# Library Requirements for RX_door1

## Required Libraries
Install these libraries in Arduino IDE:

### 1. ESP8266WiFi
- **Purpose**: WiFi connectivity for ESP8266
- **Installation**: Tools → Board → ESP8266 Boards → ESP8266WiFi
- **Version**: Latest stable version

### 2. ESP8266HTTPClient  
- **Purpose**: HTTP client for sending data to Master Hub
- **Installation**: Tools → Board → ESP8266 Boards → ESP8266HTTPClient
- **Version**: Latest stable version

### 3. WiFiClient
- **Purpose**: WiFi client functionality
- **Installation**: Tools → Board → ESP8266 Boards → WiFiClient
- **Version**: Latest stable version

## Installation Steps
1. Open Arduino IDE
2. Go to Tools → Board → ESP8266 Boards
3. Select "NodeMCU 1.0 (ESP-12E Module)" or compatible ESP8266 board
4. Install required libraries from Library Manager
5. Verify all libraries are available

## Notes
- All libraries are part of the ESP8266 core
- No external third-party libraries required
- Ensure ESP8266 board package is installed
