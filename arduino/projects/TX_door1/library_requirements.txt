# Library Requirements for TX_door1 - Dual TX Master Hub

## Required Libraries
Install these libraries in Arduino IDE:

### 1. ESPAsyncTCP
- **Purpose**: Asynchronous TCP library for ESP8266
- **Installation**: Sketch → Include Library → Manage Libraries → Search "ESPAsyncTCP"
- **Version**: 1.2.0 or later
- **Author**: me-no-dev

### 2. ESPAsyncWebServer
- **Purpose**: Asynchronous web server for ESP8266
- **Installation**: Sketch → Include Library → Manage Libraries → Search "ESPAsyncWebServer"
- **Version**: 1.2.3 or later
- **Author**: me-no-dev

### 3. EEPROM
- **Purpose**: EEPROM emulation for data persistence
- **Installation**: Built-in with ESP8266 core
- **Version**: Latest stable version

## Installation Steps
1. Open Arduino IDE
2. Go to Sketch → Include Library → Manage Libraries
3. Search for and install:
   - "ESPAsyncTCP" by me-no-dev
   - "ESPAsyncWebServer" by me-no-dev
4. Ensure ESP8266 board package is installed
5. Select "NodeMCU 1.0 (ESP-12E Module)" board

## Board Configuration
- **Board**: NodeMCU 1.0 (ESP-12E Module)
- **Upload Speed**: 115200
- **CPU Frequency**: 80 MHz
- **Flash Size**: 4M (1M SPIFFS)
- **Port**: Select appropriate COM port

## Dependencies
- ESP8266 Arduino Core (2.7.4 or later recommended)
- Arduino IDE 1.8.13 or later

## Notes
- ESPAsyncTCP and ESPAsyncWebServer are third-party libraries
- These libraries provide better performance than standard libraries
- EEPROM library is built into ESP8266 core
- Ensure all libraries are compatible with your ESP8266 board package version
